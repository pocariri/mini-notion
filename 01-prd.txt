다음의 내용들로 미니 노션 웹 서비스 PRD를 02-prd.md 파일로 만들어줘

===

- 사용자:
    - 개인 업무를 관리하고 싶은 사용자

- 사용기능:
    - 주제: 미니 노션
    - 페이지 및 기능:
        - 로그인 페이지
            - 구글로그인

        - 마이 페이지
            - 별명 변경
            - 프로필 이미지

        - 업무 페이지
            - /page 입력시 새 글 생성
            - 글 클릭시 글 상세로 이동
            - 글 상세에서 제목, 내용 입력 가능
            - 글 삭제

- 사용목적:
        - 개인용 업무 관리 툴을 직접 만들어서 무료로 사용하기 위해
        - 개인용이므로, 나에게 최적화된 기능들로 확장하기 위해
