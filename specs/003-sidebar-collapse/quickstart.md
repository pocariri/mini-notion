# Quickstart: 사이드바 접기/펼치기 검증 가이드

**Date**: 2026-07-16 | **Contracts**: [contracts/](./contracts/) | **Spec**: [spec.md](./spec.md)

구현이 끝났을 때 이 문서의 시나리오가 전부 통과해야 "완료"다(헌법 원칙 V — 테스트 통과만으로 완료 선언 금지).

## 사전 조건

- 이 워크트리(`.claude/worktrees/wt3`)에서 실행. 의존성 설치 완료 상태(`npm install` 이미 수행됨).
- 로그인 가능한 상태(Supabase 세션) 또는 기존 세션 유지 — 레일은 로그인 후 화면에서만 보인다.

## 1. 자동 테스트

```bash
npm run test:run
```

**기대 결과**: 전체 통과, 경고 0. 신규 테스트가 [store-api.md](./contracts/store-api.md) S-1~S-8과 [rail-component.md](./contracts/rail-component.md) C-1~C-9를 커버하고, 기존 52개 테스트에 회귀가 없다.

## 2. 실기동 검증

```bash
npm run dev
```

브라우저에서 `http://localhost:3000/workspace` 접속 후, 각 시나리오를 순서대로 확인한다.

## 3. 접근성 스팟 체크 (Q-11 보강)

macOS VoiceOver(⌘F5)로 토글 버튼에 도달했을 때:

- 이름이 "사이드바 접기/펼치기"로 읽히고,
- 상태가 "확장됨/축소됨(expanded/collapsed)"으로 구분되어 읽힌다 (FR-010, US3-3).
